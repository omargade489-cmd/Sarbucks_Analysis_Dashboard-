# ☕ Starbucks Beverage Analytics Dashboard

##  Project Overview

The Starbucks Beverage Analytics Dashboard is an interactive Power BI project designed to analyze nutritional and beverage-related metrics across Starbucks products. The dashboard helps users explore beverage categories, caffeine levels, calorie content, sugar content, and protein ranges through dynamic visualizations.

##  Objectives

* Analyze beverage nutritional information.
* Identify high-calorie and high-caffeine beverages.
* Compare beverage categories based on nutritional values.
* Create an interactive dashboard for business insights and decision-making.

##  Tools & Technologies

* Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Excel / CSV Dataset
* Data Modeling

## Dashboard Features

### KPI Cards

* Total Beverages
* Average Sugar
* Average Calories
* Average Caffeine

### Interactive Visualizations

* Average Calories by Beverage Category
* Starbucks Global Presence Analysis
* Beverage Category Distribution
* Average Caffeine by Category
* Top 5 Highest Caffeine Beverages
* Dynamic Slicers for Protein Range and Beverage Preparation Type

### User Experience Enhancements

* Starbucks-themed dashboard design
* Interactive filtering and drill-down capabilities
* Custom icons and branded visuals

## Data Preparation

* Imported raw beverage dataset.
* Cleaned and transformed data using Power Query.
* Removed inconsistencies and handled missing values.
* Created calculated columns and measures using DAX.
* Built an optimized data model for reporting.

## Key Insights

* Coffee beverages contain the highest average caffeine levels.
* Frappuccino categories contribute significantly to calorie intake.
* Beverage nutritional values vary considerably across preparation types.
* Users can easily identify beverages based on caffeine, sugar, calorie, and protein requirements.
